# Pokedex App

A simple Pokedex web app that allows users to search Pokémon by name and display their image using the PokeAPI.

## Features
- Search Pokémon by name
- Displays Pokémon sprite image
- Error handling for invalid names
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- PokeAPI

## How to Run
1. Download or clone the repository.
2. Open index.html in a browser.
3. Enter a Pokémon name and search.
